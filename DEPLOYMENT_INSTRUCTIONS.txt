STEP 2: LOAD DOCKER IMAGES
---------------------------
On the server, navigate to deployment directory:
  cd /opt/agastya/

Load the Docker images:
  docker load -i agastya-docker-deployment-20251126_102036.tar

Verify images loaded:
  docker images | grep react-app


STEP 3: START SERVICES
-----------------------
Start the containers in detached mode:
  docker-compose up -d

Check container status:
  docker-compose ps

View logs (if needed):
  docker-compose logs -f


STEP 4: VERIFY DEPLOYMENT
--------------------------
Check if services are running:
  curl http://localhost:8765/api/admin/features
  curl http://localhost:3456

Access the application:
  Frontend: http://your-server-ip:3456
  Backend API: http://your-server-ip:8765

================================================================================
  TROUBLESHOOTING
================================================================================

ISSUE: Port already in use
SOLUTION: 
  # Find process using port
  netstat -ano | findstr :3456
  netstat -ano | findstr :8765
  # Kill process or change ports in docker-compose.yml

ISSUE: Containers not starting
SOLUTION:
  docker-compose logs
  # Check for errors and ensure all images loaded correctly

ISSUE: Backend errors
SOLUTION:
  # Check backend logs
  docker-compose logs backend
  # Ensure config.json exists and is valid

ISSUE: Frontend can't reach backend
SOLUTION:
  # Check network connectivity
  docker network inspect react-app_agastya-network
  # Ensure backend is healthy: docker-compose ps

================================================================================
  UNINSTALL
================================================================================

To completely remove the application:

1. Stop and remove containers:
   docker-compose down -v

2. Remove images:
   docker rmi react-app-frontend react-app-backend

3. Remove deployment files:
   rm -rf /opt/agastya/
